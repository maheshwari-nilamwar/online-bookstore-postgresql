# 📚 Online Book Store Analysis Using PostgreSQL

## Project Overview

This project demonstrates database design and business analysis for an Online Book Store using PostgreSQL.

The project includes database creation, data import, SQL queries, and business insights generation.

## Dataset

### Books

* Book_ID
* Title
* Author
* Genre
* Published_Year
* Price
* Stock

### Customers

* Customer_ID
* Name
* Email
* Phone
* City
* Country

### Orders

* Order_ID
* Customer_ID
* Book_ID
* Order_Date
* Quantity
* Total_Amount

## SQL Concepts Used

* SELECT
* WHERE
* GROUP BY
* HAVING
* ORDER BY
* JOINS
* Aggregate Functions
* DISTINCT

## Business Questions Solved

* Total Revenue Analysis
* Genre-wise Book Sales
* Most Frequently Ordered Book
* Customer Spending Analysis
* Inventory Tracking
* Author-wise Sales Analysis
